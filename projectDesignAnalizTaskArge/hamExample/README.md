# hamExample

Sistemin bütününü ilgilendiren ham örnek materyallerin düşürüldüğü alan.

## Temel Kural

**Bu klasöre dosyaları HEP KULLANICI koyar.** Claude buraya dosya eklemez.

## Kullanım

- Bulduğun her çeşit dökümanı (screenshot, pdf, link listesi, rakip ürün, tasarım notu...) buraya at.
- Sıralama veya proje ataması zorunlu değil — `example1`, `example2` gibi adlandır.
- Design ile ilgili agent (tasarimci / arge_muhendisi / analiz_uzmani) oturum sonunda "projeye mi devam, yoksa hamExample analizi mi?" diye sorar.
- Analiz seçilirse agent dosyaları listeler, analiz eder, sonucu ilgili projenin `example_design_duzenlenmis/` klasörüne yazar.

## Yapı

Düz/flat — alt klasör yok, dosyalar doğrudan buradadır.
