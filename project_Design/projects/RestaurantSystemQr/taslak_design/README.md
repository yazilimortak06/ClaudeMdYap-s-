# Taslak Design — RestaurantSystemQr

Kağıda, Paint'e veya herhangi bir araca çizilen ham taslaklar.
Foto, screenshot veya açıklama md dosyası olarak buraya gelir.

## Ne Gelir?
- Kağıt üzerine çizilmiş ekran taslakları (fotoğraf)
- Paint / MS Paint çizimleri (PNG, JPG)
- Hızlı karalama açıklamaları (.md)

## Dosya Adlandırma
`YYYY-MM-DD_<ekran_veya_konu>_taslak.png / .jpg / .md`
Örn: `2026-05-14_qr_okuma_ekrani_taslak.jpg`

## Akış
Buradaki taslaklar → tasarımcı veya arge tarafından incelenir →
`example_design_duzenlenmis/` veya `mevcut_design/` a taşınır.
