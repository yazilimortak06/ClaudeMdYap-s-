# The Soul Pub — QR Menü Teknik Dokümantasyonu
## Platform: Pardon App (webmenu.pardonapp.co)

---

## 1. Genel Bakış

The Soul Pub, Pardon App'in SaaS QR Menü altyapısını kullanan Ankara merkezli premium bir bar/pub işletmesidir. Müşteriler masadaki QR kodu okutarak dijital menüye erişir ve sipariş verebilir.

**Analiz Edilen URL:** https://webmenu.pardonapp.co/soul-pub
**Platform:** Pardon App — "Restoranlar için #1 AI"

---

## 2. Platform Mimarisi

### 2.1 Frontend Yapısı

```
Framework   : Next.js (React tabanlı, SSR destekli)
Dil         : TypeScript
Stil        : Tailwind CSS
Mobil       : Responsive / Mobile-first
Tema        : Dark mode (Premium koyu tema)
Deployment  : CDN ile dünya genelinde dağıtım
Güvenlik    : Cloudflare WAF (bot engeli aktif)
```

### 2.2 Backend Tahminleri

```
API         : RESTful veya GraphQL
Auth        : JWT token tabanlı
DB          : PostgreSQL (tenant bazlı)
Cache       : Redis
Media       : CDN üzerinden görsel servisi
AI Engine   : OpenAI API veya özel LLM
```

### 2.3 Multi-Tenant SaaS Yapısı

```
Tenant = İşletme (The Soul Pub)
Slug   = soul-pub (URL anahtarı)
Branch = Şube (Kavaklıdere şubesi)
Menu   = Aktif menü (yiyecek menüsü)
```

## 9. Önemli Notlar

1. **Site Erişim Kısıtlaması:** webmenu.pardonapp.co bot erişimini engelliyor (HTTP 403).
   Gerçek tarayıcı ile erişildiğinde JavaScript yükleniyor ve içerik görünüyor.

2. **Menü Güncelliği:** Menü fiyatları dinamik olduğundan zaman içinde değişebilir.
   Bu dokümandaki fiyatlar analiz tarihine ait anlık veridir.

3. **İçecek Menüsü:** Soul Pub'ın 200+ viski dahil geniş içecek menüsü
   webmenu.pardonapp.co üzerindeki QR menüde ayrı olarak bulunuyor olabilir.
   Bu analizde sadece yiyecek kategorileri erişilebildi.

4. **AI Özellikleri:** Pardon AI modülü aktif kullanımda.
   Platform kendini "Restoranlar için #1 AI" olarak konumlandırıyor.
